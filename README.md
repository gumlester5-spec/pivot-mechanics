# Pivot Mechanics - App de Gestión de Taller de Motos

Esta aplicación está diseñada para gestionar clientes y reparaciones en un taller de mecánica de motocicletas.

## Características Principales

- **Vista del Cliente (El Rastreador)**: Permite a los clientes ver el estado de su moto en tiempo real (Recepción, Diagnóstico, Repuestos, Listo).
- **Vista del Mecánico (Lista de Trabajo)**: Gestión de tareas asignadas y actualización de estados.
- **Vista del Administrador (Centro de Control)**: Registro de nuevas órdenes, asignación de mecánicos y visión general del taller.

## Tecnologías

- React + TypeScript + Vite
- Supabase (Base de datos y autenticación)
- CSS Modules / Vanilla CSS para estilos

## Instalación

1. Clonar el repositorio.
2. Ejecutar `npm install`.
3. Ejecutar `npm run dev` para iniciar el servidor de desarrollo.
